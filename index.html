"Сайтты қалай жасадым?"

1. GitHub Репозиторий

Алдымен GitHub-та репозиторий жасадым. Оған барлық HTML және CSS файлдарды жүктедім.

2. Кодтау

HTML және CSS қолдана отырып, басты бет пен портфолио бетінің құрылымын жасадым.

3. GitHub Pages

GitHub Pages арқылы сайтты онлайн жарияладым.

4. Дизайн
